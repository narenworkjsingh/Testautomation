This automation code is written to test the counter app
Below setup is required to run this code
1) Download selenium chrome driver and keep it in below foder C:\\SeleniumDrivers
2) Create "Testcheck" and "Report" folder under c driver "C:/Testcheck/reports" or check the folders when download the reports
3) Run the command to go to venvTest enviroment and activate the enviroment
4) Run "pytest -sv driver\test_cases.py --browser "chrome" to execute the code.
